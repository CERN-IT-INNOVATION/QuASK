Our Purpose

What differentiates our software from anything else?

Complex features w/ little code

We let researchers create complex machine learning scenarios within tens of lines of code, providing a very high-level API for that.

Reference implementation

We know how much time you can waste replicating the results from the latest papers. QuASK is also used as a reference implementation for the most popular papers.

Pedagogical mission

You can use the set of introductory tutorials to start your journey in quantum machine learning with quask.
